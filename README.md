# Social Collab Form

A minimal, responsive React form that collects user information and submits it to an API endpoint.

## Features

- Responsive design with custom color scheme
- Form validation
- API integration for form submission
- Exclusive code display with copy functionality
- Automatic redirect after code copy
- Banner image support

## Setup

1. Install dependencies:
```bash
npm install
```

2. The form is configured to submit to the API endpoint defined in `src/utils/api.js`. Update the `API_URL` constant if needed.

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Color Scheme

- Background: #F2F2F2
- Accent: #EAE4D5
- Secondary: #B6B09F
- Text: #000000
